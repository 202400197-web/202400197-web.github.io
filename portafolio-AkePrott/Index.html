App de Gestión de Tareas
Descripción

Aplicación full-stack para gestionar tareas en tiempo real. Incluye autenticación con Firebase, drag & drop para reorganizar tareas, notificaciones push y modo oscuro. Ideal para equipos pequeños o uso personal.

Tecnologías utilizadas
  • Frontend: React, Tailwind CSS, React Beautiful DnD
  • Backend: Firebase (Auth, Firestore, Realtime Database)
  • Otras: Vite, Context API
Características principales
  • Sincronización en tiempo real entre dispositivos
  • Arrastrar y soltar tareas
  • Registro e inicio de sesión seguro
  • Diseño responsive y accesible
